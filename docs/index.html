Frequently Asked Questions

Is RetailAgent free to use? +

Yes! RetailAgent is completely free. You only need a free Gemini API key from Google.

Is my data safe? +

Absolutely! All data is stored locally on your device. We don't collect or transmit any personal information.

Which platforms are supported? +

Currently supported: Amazon, Flipkart, eBay, and Walmart. More platforms coming soon!

Can I use it without an API key? +

No, an API key is required for the AI to understand your shopping intents. But it's free and easy to get!

Does it work on mobile? +

Currently, RetailAgent is a Chrome extension for desktop. Mobile support is planned for the future.
